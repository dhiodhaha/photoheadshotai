# Project Overview: Photoheadshot

Photoheadshot is a full-stack web application designed for generating AI headshots. It uses modern AI generation pipelines (fal.ai, R2 for storage) and provides a user-facing interface for photo uploads, credit management, and image browsing.

## Core Purpose
- AI-powered headshot generation.
- User credit system (purchase, redemption, coupons).
- Integrated photo gallery and job tracking.
