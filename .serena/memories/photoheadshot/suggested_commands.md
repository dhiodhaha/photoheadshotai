# Suggested Commands: Photoheadshot

- **Development**: `pnpm dev`
- **Build**: `pnpm build`
- **Database Studio**: `pnpm db:studio` (for exploring the data).
- **Prisma Push**: `pnpm db:push` (for schema updates).
- **Linting/Fixing**: `pnpm lint:fix` (using Biome).
- **Formatting**: `pnpm format` (using Biome).
- **Testing**: `pnpm test` (Vitest).
