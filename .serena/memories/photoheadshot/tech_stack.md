# Tech Stack: Photoheadshot

- **Framework**: TanStack Start (React + Nitro + TanStack Router).
- **Core Logic**: TypeScript.
- **Styling**: Tailwind CSS v4.
- **Database**: Prisma + PostgreSQL (via `pg`).
- **Auth**: Better Auth.
- **Storage**: Cloudflare R2 (via AWS SDK).
- **AI Integration**: fal.ai.
- **Tooling**: Biome (formatting/linting), Vitest (testing), Vite (build runner).
